Это упрощённая версия AI Flow. 

Суть: у нас есть AI-агенты, у каждого есть бюджет токенов, и мы можем связывать их в цепочки.

/agents - Список агентов, создание/удаление 

/flow - Визуальный редактор графа агентов

Мини-платформа для управления AI-агентами и визуального построения потоков автоматизации

![Demo](./my_project.gif)

---




- Создание и управление AI-агентами с бюджетом токенов
- Валидация данных через Zod
- Визуальный редактор графов (@xyflow/react)
- Реактивное состояние через Valtio (Proxy)
